#运算符

##运算符

规则：

+   如果有一个操作数复数，另一个操作数先转换为复数
+   否则，如果有一个操作数是浮点数，另一个操作数先转换为浮点数
+   否则，如果有一个操作数是长整数，另一个操作数先转换为长整数
+   否则，两者都是普通整数，无需转换

##算术运算符

```
-

%
** 幂
// 整除，舍去小数部分
```
