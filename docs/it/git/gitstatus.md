#git status

追踪文件状态

##git文件状态

文件状态：

+   未追踪
    
    新建的文件都是未追踪状态

+   已追踪
    
    通过git add命令把文件变为已追踪状态

已追踪文件状态：

+   已修改(modified)
    
    文件被修改，但未进入暂存区

+   已暂存(staged)
    
    通过git add命令将已修改文件放入暂存区

+   已提交(comitted)
    
    通过git commit命令将暂存区的文件提交到本地数据库

对于已追踪文件，每次修改后都需要通过git add命令将文件放入暂存区
